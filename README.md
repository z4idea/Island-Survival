# 孤岛求生 · Island Survival

类 Hades 俯视角群岛探索生存游戏。程序化生成的群岛（320×320，主岛 + 多座离岛）上采集资源、狩猎野兽、在篝火旁存档与强化，购买小木舟远渡深海，最终挑战随机盘踞在某座岛屿山顶的「岛屿之王 · 远古巨熊」。

## 技术栈

- **TypeScript + Vite** — 开发与构建
- **PixiJS 8** — WebGL 渲染（全程序化矢量图形，零美术素材）
- **Rapier 2D**（`@dimforge/rapier2d-compat`）— 物理引擎：刚体移动、碰撞、击退、深水边界
- **WebAudio** — 程序化合成音效（零音频素材）

## 运行

```bash
npm install
npm run dev      # 开发服务器（端口 5188）
npm run check    # TypeScript 静态检查
npm run build    # 产物构建
```

| 按键 | 功能 |
| --- | --- |
| `W A S D` | 移动 |

## 玩法要点

- **篝火 = 存档点**：休息**只保存进度**（localStorage）并把该篝火设为复活点，**不会回血** —— 恢复生命只能靠进食（生食/熟食）、嗜血天赋和生命上限升级；死亡后在最后休息的篝火满血复活。
- **资源**：木材 🪵、石块 🪨、浆果 🫐、生肉 🍖、兽皮 🐾 —— 来自树木、岩石、灌木与猎物。
- **进食与烹饪**：生食是劣质应急口粮 —— 生浆果只回 4 血，生肉只回 7 血且**约 30% 概率食物中毒 🤢**（生食所致，持续轻微掉血，只能靠熟食或时间解除，区别于蛇毒）。在篝火打开「🍳 烹饪」，消耗生鲜 + 1 木柴当燃料烤成熟食带在身上（出征前备粮）：🍗 **烤肉**（回 22，安全口粮，F 吃）、🍓 **莓果干**（回 10 + 回耐力，Q 吃）、🍢 **烤肉串**（回 18 + 30 秒攻击 +20%，R 吃）、🥘 **兽肉炖锅**（回 15 + 10 秒持续回血，R 吃）。Q/F 在有熟食时优先吃安全熟食，断粮才退而吃生食。
- **树上猴子**：约 8% 的普通树会露出摆动的猴子尾巴。身体触碰树后，猴子会随机偷走一种当前持有的资源或货币的 10%（向上取整）并快速逃跑；在它跑远消失前击杀，可夺回全部赃物。
- **强化**：在篝火菜单消耗资源升级 攻击 / 生命上限 / 耐力上限。
- **钱币与商店**：击杀动物概率掉落 银币●（常见）/ 金币●（少见）/ 钻石◆（稀有，Boss 必掉一批）。在篝火打开「🛒 神秘商人」：购买 5 种新武器（战斧/双刃/重锤/劲弩/烈焰剑）、5 种天赋（拾荒者/幸运/疾风/嗜血/坚韧）、7 种武器皮肤（鎏金/翡翠/血色/寒冰/星辉/幽冥/雷光，均带粒子光效，攻击迸发 + 待机微光），每把武器还可逐级升级（+12% 伤害/级，最高 5 级）。武器栏随拥有的武器扩展（1~8 数字键切换）。
- **生态**：岩蟹（沙滩）、野猪（会冲锋）、灰狼（夜晚更快更凶、群体仇恨）、麋鹿（被动逃跑）、毒蛇（草丛突袭，攻击附带**中毒**持续掉血，进食可解毒）、岩山羊（山坡栖息，平时中立，被攻击才用羊角反击）、海鸥（沙滩飞行生物，会飞过水面逃跑，适合练弓）、**猛虎**（森林顶级掠食者：速度快、伤害高、会扑击）。
- **海洋生态**：游鱼（无害，被动逃跑）、海龟（中立，被攻击才反击）、巨鲨（攻击性，会在水中突袭冲撞）——全部只在水中活动。
- **水与船**：在水中停留超过 5 秒开始溺水（每秒 -4 生命）。在商店购买 🛶 小木舟后靠近水面自动乘船：不再溺水、航速提升，且可以穿越深海前往其他岛屿。
- **战争迷雾**：小地图只显示探索过的区域，未到过的地方被迷雾遮挡，篝火与 Boss 标记需探索后才会显现。迷雾随存档保存。
- **洞穴探险**：群岛上散布着 3 座洞穴（岩地上的黑暗洞口），按 `E` 进入。洞内一片漆黑（只照亮身边），有 **宝箱**（开出随机数量的银币/金币/钻石，数量随洞穴大小 1~3 个）、**水晶矿脉**（敲击掉钻石，敲碎必出）。已开宝箱与已采水晶随存档保存。在出口光柱处按 `E` 离开。低概率出现 **宝藏洞穴**：没有任何野兽，金砂铺地，5~6 个宝箱且钻石必出。
- **洞穴守卫**：**吸血蝙蝠** —— 飞行极快、近战打不到，唯一反制是在被发现前用弓弩射杀（短弓 3 箭）；一旦贴身就会咬住你的头持续吸血（共 20 点，吸饱消失，多只可叠加）。**妖狐** —— 属性近似猛虎，攻击附带 **魅惑**：移动方向短暂颠倒。
- **神器祝福**：开局约一分钟后，某座岛屿上会随机洒下一道**天降白色光柱**（小地图上有 ✦ 白色标记，无视迷雾可见）。走入圣光按 `E` 举行祝福仪式，随机获得一件尚未拥有的神器；领取后过几分钟神光会再次降临，直到集齐全部 3 件。神器不在商店出售，但神器武器可以在商店升级。
  - 💘 **丘比特的弓**（武器）：爱神的金弓，被爱心箭射中的动物有 10% 概率**坠入爱河**——被粉色爱心环绕、永不攻击你（再次攻击它会心碎清醒，Boss 免疫）。
  - 🪄 **阿比努斯的权杖**（武器）：手持时身周浮现青绿色**施法领域环**，在领域内点击鼠标即可在落点召唤**冥火**：法阵聚集后爆发，灼烧范围内的所有敌人（附带持续燃烧）。
  - ⚡ **宙斯的雷霆神矛**（武器，长矛形态）：矛身是一道缭绕电芒的金色闪电。命中的动物会被**天降闪电**劈中追加伤害；**雨天**电闪雷鸣，化作威力更大、带链式溅射的**大型闪电**。
  - 🕊️ **大天使的翅膀**（挂件，新系列：被动饰品）：背后常驻一对圣翼，翻滚化作**圣翼冲刺**——高速掠地、无敌帧更长，**无视树木与岩石直接穿行而过**，并撞伤冲刺路径上的一切生物。
- **状态效果**：血条旁会显示当前状态图标 —— ☠️ 中毒（进食可解）、🩸 流血（蝙蝠吸血中）、💫 魅惑（WASD 反向）、💧 溺水。
- **昼夜循环**：5 分钟一昼夜，夜晚视野变暗、狼群仇恨范围扩大。
- **天气**：会随机下雨（右上角时钟变为 🌧️）。雨天画面变暗、有雨幕和雨声，**玩家移动速度 -20%**，雨停后恢复。
- **野兽成长**：随累计游戏时长，新出生的动物会越来越强 —— 每分钟血量与攻击 +2%（速度按三折增速），上限 +150%；Boss 巨熊按半速成长。游戏时长随存档保存，越拖越危险，尽早强化自己。
- **终极目标**：击败中央山顶的远古巨熊（冲锋 + 重击 + 低血量狂暴）。
