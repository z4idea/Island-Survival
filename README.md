# 孤岛求生 · Island Survival

类 Hades 俯视角群岛探索生存游戏。程序化生成的群岛（240×240，主岛 + 多座离岛）上采集资源、狩猎野兽、在篝火旁存档与强化，购买小木舟远渡深海，最终挑战随机盘踞在某座岛屿山顶的「岛屿之王 · 远古巨熊」。

## 技术栈

- **TypeScript + Vite** — 开发与构建
- **PixiJS 8** — WebGL 渲染（全程序化矢量图形，零美术素材）
- **Rapier 2D**（`@dimforge/rapier2d-compat`）— 物理引擎：刚体移动、碰撞、击退、深水边界
- **WebAudio** — 程序化合成音效（零音频素材）

## 运行

```bash
npm install
npm run dev      # 开发服务器（端口 5188）
npm run check    # TypeScript 静态检查
npm run build    # 产物构建
```

## 玩法要点

- **篝火 = 存档点**：休息可回满血并写入存档（localStorage），死亡后在最后休息的篝火复活；休息会让岛上野兽重生。
- **资源**：木材 🪵、石块 🪨、浆果 🫐、生肉 🍖、兽皮 🐾 —— 来自树木、岩石、灌木与猎物。
- **强化**：在篝火菜单消耗资源升级 攻击 / 生命上限 / 耐力上限。
- **钱币与商店**：击杀动物概率掉落 银币●（常见）/ 金币●（少见）/ 钻石◆（稀有，Boss 必掉一批）。在篝火打开「🛒 神秘商人」：购买 5 种新武器（战斧/双刃/重锤/劲弩/烈焰剑）、5 种天赋（拾荒者/幸运/疾风/嗜血/坚韧）、4 种武器皮肤（鎏金/翡翠/血色/寒冰），每把武器还可逐级升级（+12% 伤害/级，最高 5 级）。武器栏随拥有的武器扩展（1~8 数字键切换）。
- **生态**：岩蟹（沙滩）、野猪（会冲锋）、灰狼（夜晚更快更凶、群体仇恨）、麋鹿（被动逃跑）、毒蛇（草丛突袭，攻击附带**中毒**持续掉血，进食可解毒）、岩山羊（山坡栖息，平时中立，被攻击才用羊角反击）、海鸥（沙滩飞行生物，会飞过水面逃跑，适合练弓）、**猛虎**（森林顶级掠食者：速度快、伤害高、会扑击）。
- **海洋生态**：游鱼（无害，被动逃跑）、海龟（中立，被攻击才反击）、巨鲨（攻击性，会在水中突袭冲撞）——全部只在水中活动。
- **水与船**：在水中停留超过 5 秒开始溺水（每秒 -4 生命）。在商店购买 🛶 小木舟后靠近水面自动乘船：不再溺水、航速提升，且可以穿越深海前往其他岛屿。
- **战争迷雾**：小地图只显示探索过的区域，未到过的地方被迷雾遮挡，篝火与 Boss 标记需探索后才会显现。迷雾随存档保存。
- **昼夜循环**：5 分钟一昼夜，夜晚视野变暗、狼群仇恨范围扩大。
- **终极目标**：击败中央山顶的远古巨熊（冲锋 + 重击 + 低血量狂暴）。

## 代码结构

```
src/
  main.ts              入口：标题画面与 UI 接线
  game.ts              主控：物理世界、战斗结算、篝火/存档、昼夜、镜头
  defs.ts              全局常量与武器/动物/升级数据定义
  core/   input.ts  audio.ts  save.ts
  world/  worldgen.ts（程序化岛屿） worldrender.ts（区块渲染）
  entities/ player.ts  animals.ts  drops.ts  projectiles.ts
  ui/     hud.ts（血条/小地图/菜单 DOM）
  fx.ts                粒子与伤害飘字
  utils/  noise.ts     值噪声 FBM 与确定性随机
```
