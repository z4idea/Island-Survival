# 海神三叉戟卷浪特效设计

## 目标

将波塞冬三叉戟现有的白色通用斩击和零散圆点水沫，替换为清晰的蓝色海浪表现：

- 攻击时形成 180 度双层卷浪。
- 持有三叉戟在海中移动时，玩家两侧持续卷起海水。
- 不改变攻击数值、范围、移动速度、碰撞或存档行为。

## 视觉语言

- 外层主浪使用深海蓝，体现体积和力度。
- 内层浪脊使用亮蓝，强化横扫方向。
- 浪尖使用少量浅蓝泡沫高光，不使用纯白色。
- 待机时只保留低密度蓝色水沫，避免抢过攻击和移动特效。

## 攻击效果

三叉戟攻击沿现有 180 度挥舞动画播放专属卷浪：

- `slashG` 不再使用皮肤的通用白色斩击。
- 外浪和内浪采用两条有厚度的弧形带。
- 泡沫沿外侧浪尖分段绘制，形成不连续的浅蓝波峰。
- 卷浪随攻击进度从小到大展开，并在攻击结束后快速淡出。
- 特效朝向跟随攻击时锁定的瞄准方向，不改变命中判定。

## 海中移动效果

玩家手持三叉戟且位于水面、存在移动输入时：

- 玩家左右两侧各绘制一股向后卷起的弧形浪翼。
- 浪翼方向依据实际移动速度旋转，始终朝运动方向后方展开。
- 浪体通过相位脉动表现卷动，移动停止或离开水面后立即淡出。
- 保留低频蓝色水滴粒子作为飞沫，但不再把圆点粒子作为主体。

## 实现边界

- 只修改 `src/entities/player.ts`。
- 新增三叉戟攻击浪与踏浪 Graphics，不扩展公共 `Particles` API。
- 复用现有 `swingT`、`aim`、速度和 `seaWalking` 状态。
- Graphics 加入玩家根容器，攻击浪位于角色后方，泡沫和浪脊保持可见。
- 不新增资源文件、第三方依赖或存档字段。

## 验证

- 执行 `npm run check`，要求 TypeScript 静态编译零错误。
- 源码检查确认三叉戟不再走通用白色 `slashG` 配色。
- 源码检查确认海中移动卷浪仅在 `seaWalking` 且移动时显示。
